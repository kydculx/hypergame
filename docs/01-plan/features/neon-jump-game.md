# PDCA Plan: 네온 점프 (Neon Jump)

## 1. 개요 (Overview)
하이퍼 캐주얼 게임의 클래식인 "Flappy Bird" 스타일의 러닝 액션 게임을 개발합니다. 네온 스타일의 비주얼과 단순한 원터치 조작으로 중독성을 유도합니다.

## 2. 목표 (Goals)
- **원터치 조작**: 화면 터치/클릭 시 캐릭터 점프.
- **무한 러닝**: 장애물을 피하며 최대한 멀리 가는 것이 목표.
- **시각적 매력**: 어두운 배경에 빛나는 네온 컬러 오브젝트 (Cyberpunk 느낌).

## 3. 핵심 기능 (Key Features)
- **플레이어 물리**: 중력 적용 및 점프 가속도 구현.
- **장애물 생성**: 일정 간격으로 위/아래 파이프(또는 기둥) 생성.
- **충돌 처리**: 플레이어가 장애물이나 바닥/천장에 닿으면 게임 오버.
- **점수 시스템**: 장애물을 통과할 때마다 점수 획득 및 플랫폼 전송.

## 4. 기술 스택 (Technical Stack)
- **Engine**: HTML5 Canvas (2D). Three.js보다 가볍고 2D 게임에 적합.
- **Audio**: Web Audio API (점프 소리, 득점 소리, 충돌 소리).
- **Style**: Canvas API의 `shadowBlur` 속성을 활용한 네온 효과.

## 5. 로드맵 (Roadmap)
1. **Phase 1: 게임 루프 및 물리 엔진** - 중력, 속도, 위치 업데이트 로직.
2. **Phase 2: 렌더링 시스템** - 네온 스타일의 플레이어 및 장애물 그리기.
3. **Phase 3: 장애물 관리** - 생성, 이동, 삭제(화면 밖) 로직.
4. **Phase 4: 충돌 및 점수** - AABB 충돌 검사 및 점수 카운팅.
5. **Phase 5: 오디오 및 플랫폼 연동** - 효과음 추가 및 `postMessage` 연동.

## 6. 성공 지표 (Success Metrics)
- 부드러운 60fps 애니메이션.
- 정확한 히트박스 판정.
- 플랫폼 내 정상 구동 및 점수 기록.
