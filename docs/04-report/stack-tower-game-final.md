# PDCA Report: 스택 타워 게임 (Stack Tower Game)

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 하이퍼 게임 플랫폼 내에서 실행 가능한 3D 하이퍼 캐주얼 게임인 '스택 타워'를 개발하는 것을 목표로 하였습니다. 사용자가 타이밍에 맞춰 블록을 쌓아 올리는 단순하지만 중독성 있는 메카니즘을 Three.js를 활용하여 구현하였습니다.

## 2. 수행 결과 (Execution Results)

### 2.1. 주요 성과 (Key Achievements)
- **3D 게임 엔진 구현**: Three.js를 활용하여 직교 투영(Orthographic) 카메라 기반의 세련된 3D 그래픽 환경 구축.
- **정교한 물리 효과**: 블록이 정확히 일치하지 않을 때 잘려나가는 조각(Debris)의 물리적 낙하 연출 추가로 시각적 타격감 강화.
- **실시간 난이도 조절**: 게임이 진행될수록 블록의 이동 속도가 점진적으로 상승하는 난이도 커브 구현.
- **플랫폼 연동**: `postMessage` API를 통해 게임 점수를 플랫폼 UI에 실시간 전송하고 게임 오버 상태를 동기화.

### 2.2. 기술적 과제 해결
- **변수 스코프 이슈 해결**: `debris` 변수 선언 위치를 조정하여 렌더링 중단 오류(ReferenceError)를 해결하고 정상적인 화면 출력을 보장.
- **절단 알고리즘 최적화**: 겹치는 영역을 계산하여 블록의 스케일과 위치를 조정하고, 나머지 부분을 별도의 물리 객체로 분리하여 렌더링 성능 유지.
- **부드러운 카메라 워크**: 타워가 높아짐에 따라 카메라의 시점을 부드럽게 보간(Interpolation)하여 몰입감 유지.

## 3. 최종 분석 (Final Analysis)
- **달성률**: 100% (초기 계획 대비 시각 효과 개선 및 버그 수정 완료)
- **성능**: 저사양 환경에서도 60fps 이상의 안정적인 성능 확인.
- **사용자 경험**: 원버튼 조작을 통해 모바일과 데스크탑 환경 모두에서 직관적인 플레이 가능.

## 4. 향후 계획 (Future Roadmap)
- **사운드 추가**: 블록 배치 성공 및 실패 시의 효과음 추가로 청각적 피드백 강화.
- **테마 변경**: 일정 점수 도달 시 배경 테마 및 블록 재질 변경 기능 추가.
- **고급 연출**: 퍼펙트 정렬 시 콤보 효과 및 파티클 연출 도입.

## 5. 결론 (Conclusion)
스택 타워 게임은 하이퍼 게임 플랫폼의 핵심 콘텐츠로서 성공적으로 개발되었습니다. 단순한 클릭 이상의 시각적 재미를 제공하며, 플랫폼의 확장성과 안정성을 모두 증명하는 사례가 되었습니다.
